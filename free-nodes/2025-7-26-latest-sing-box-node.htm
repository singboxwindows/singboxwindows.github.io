Quantumult中添加节点的方法详解

方法一：导入订阅链接（推荐）

  1. 打开Quantumult X

  2. 点击右下角【设置】

  3. 进入【配置】>【服务器】

  4. 点击右上角“+”选择“通过URL导入”

  5. 粘贴订阅地址，命名并保存

  6. 返回首页，点击节点列表选择一个服务器即可启用

方法二：手动添加节点

  1. 进入【设置】>【配置】>【服务器】

  2. 点击右上角“+”，选择节点类型（如Shadowsocks）

  3. 输入服务商提供的服务器、端口、加密方式、密码等信息

  4. 命名并保存
