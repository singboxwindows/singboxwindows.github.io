在网络环境愈发复杂、信息获取受到诸多限制的今天，一款强大的网络工具变得尤为重要。Quantumult X，这款专为 iOS 设计的全功能网络工具，已悄然成为众多“进阶用户”绕过限制、优化网络体验的利器。它不仅可以实现科学上网，还支持高级的分流规则、流量统计、策略组配置，甚至在视频流平台提速方面也有不俗表现。

如果你刚刚听说 Quantumult X，或者已经安装却不知道该如何配置与使用，这篇文章将为你提供最详尽的使用指南。我们将从下载安装开始，逐步解析其基本配置、进阶功能、规则自定义方法，并解答使用中可能遇到的问题。文章不仅适合初学者，也欢迎老用户一边阅读一边回顾自己是否配置到位，发掘更多潜能。

一、什么是 Quantumult X？

Quantumult X 是一款专注于网络规则分流与代理管理的高级工具。它的核心用途是：科学上网、网络加速、规则控制、流量监控、代理链路管理等。由于其强大的灵活性和稳定性，在 iOS 用户圈中广受推崇，尤其适合中高级用户或网络安全爱好者。

它不同于一般的 VPN 客户端，它更像一个小型的可编程网络路由器，你可以在里面设定谁走代理、谁走直连，甚至为不同的流量制定多级规则。这种“规则优先”的架构，让它在应对复杂网络环境时游刃有余。

二、为什么选择 Quantumult X？

在 iOS 平台上，类似的工具还有 Shadowrocket、Loon、Stash 等，但 Quantumult X 能脱颖而出，靠的正是其细腻的功能配置能力和稳定性极高的运行机制。

2.1 核心优势

  • 策略组支持：可以设置多个策略组（如自动选择、节点列表、负载均衡等）。

  • 脚本支持：通过 JavaScript 执行高级功能（如重写请求、广告拦截等）。

  • 流量分析：按时间、应用、节点等多维度进行网络使用统计。

  • 规则灵活：内建强大的“规则分流系统”，用户可轻松自定义规则优先级。

  • 轻量美观：UI 简洁，几乎无卡顿，后台运行稳定。

三、如何安装 Quantumult X？

Quantumult X 目前仅限于 iOS 平台，且不提供免费试用，属于一次性买断制。

3.1 安装步骤

  1. 打开你的 iPhone 或 iPad，进入 Apple App Store；

  2. 搜索 “Quantumult X”；

  3. 点击“购买”或“获取”按钮；

  4. 安装完成后，首次打开会提示设置 VPN 配置文件，点击允许。

四、配置指南：从零开始配置 Quantumult X

Quantumult X 虽然功能强大，但上手并不复杂，遵循下面的步骤即可完成初始配置：

4.1 导入配置文件

Quantumult X 不像某些工具那样自动生成配置文件，你需要从服务商那里获取一份配置文件，形式通常为：

  • .qxconfig 文件（可以直接导入）；

  • 或者一段 订阅链接（Surge 格式或 Quantumult X 格式均可）。

方法一：扫码导入

  1. 打开 Quantumult X；

  2. 点击右下角“设置”；

  3. 选择“配置” > “下载”；

  4. 点击右上角“扫描二维码”，扫描你的服务提供商给的订阅码。

方法二：复制链接导入

  1. 将订阅链接复制；

  2. 返回 Quantumult X；

  3. 在“配置”中粘贴链接，点击“下载配置”。

导入后，建议点击右上角“生效”，确保配置已经激活。

五、规则与策略组：玩转分流的关键

Quantumult X 的灵魂，在于分流规则与策略组配置。这是它与普通 VPN 最大的差异。

5.1 什么是策略组？

策略组是一个“逻辑集合”，用来指示某类请求要使用哪种连接方式。常见的策略组包括：

  • 直连（DIRECT）：不通过代理，直接连接；

  • 全局代理（PROXY）：所有请求走代理节点；

  • 节点列表：列出多个节点供用户选择；

  • 自动选择（AUTO）：根据延迟或负载自动切换节点；

  • 拒绝（REJECT）：拦截特定流量，常用于广告屏蔽。

5.2 自定义规则分流

你可以根据域名、IP、应用名等创建详细规则。例如：

plaintext
DOMAIN-SUFFIX,google.com,PROXY
DOMAIN-SUFFIX,baidu.com,DIRECT
PROCESS-NAME,YouTube,PROXY

这些规则将自动应用在访问过程中，实现精准控制。

✅ 小贴士：建议使用“规则分组 + GEOIP + DOMAIN”组合，可以最大程度地控制代理策略，防止误杀或漏判。

Quantumult X 不止于分流和科学上网，它还有许多令人惊喜的功能：

6.1 流量统计

  • 查看每个应用程序的流量消耗；

  • 区分本地直连和代理流量。

6.2 日志记录

  • 实时查看请求记录；

  • 追踪每一个请求命中的是哪条规则；

  • 方便排错与优化规则配置。

6.3 重写与脚本

  • 修改 HTTP 请求/响应内容；

  • 实现自动签到、广告屏蔽、内容改写等功能；

  • 可加载用户脚本或自定义编写 JavaScript 脚本。

七、使用中常见问题与解决方案

7.1 Quantumult X 无法连接？

  • 检查 VPN 权限是否授权；

  • 确保节点可用（ping 不通的换个节点）；

  • 检查配置文件是否正确生效。

7.2 VPN 状态频繁断开？

  • 避免开启低功耗模式；

  • 设置中允许后台活动；

  • 尽量使用稳定节点，不建议使用公共免费节点。

7.3 如何重置 Quantumult X？

在设置页中选择“恢复出厂设置”，即可清除所有配置，回归初始状态。

7.4 能否在一个设备上安装多个版本的 Quantumult X？

官方并未推出多版本，因此只能安装一个。若需切换配置，建议保留多个 .qxconfig 文件，根据场景导入即可。

八、进阶建议：如何玩出 Quantumult X 的极致体验？

  • 学习正则表达式与规则优先级；

  • 利用节点测速功能，优化连接质量；

  • 导入社区共享规则（如ACL4SSR）提升体验；

  • 利用脚本拓展功能，自定义交互体验，如签到脚本、广告清洗等。
